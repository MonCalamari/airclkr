airclkr//
A dedicated, low-jitter Ableton Link to MIDI Clock bridge for monome norns.

About
airclkr// provides phase-accurate sync bridge between your Ableton Link session and your external MIDI hardware. Built by MonCalamari for live performance, it prioritizes timing stability and CPU efficiency.

Note: This script was developed with the assistance of an AI to structure and optimize the code for the norns environment.

Features
Phase-Locked Transport
MIDI Panic (K2+K3)
Optimized for CPU efficiency (15 FPS UI)
Positive Offset Compensation (0-100ms)

Installation
Install via the norns.community catalog or directly via GitHub.
